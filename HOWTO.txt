This project contains a maven project with showing an example of
Cucumber Framework, Cucumber-jvm and Java Language for the glue code.
In order to run this project you need maven installed on your machine
(see more on http://maven.apache.org/). Then run the following command
on a command prompt located at the root directory of the project (where
pom.xml file is).
mvn clean test
